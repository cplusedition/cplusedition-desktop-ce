Introduction
An offline notes taking app that creates HTML/CSS documents. It help you create interactive, rich media, rich text documents. It also help you with todo lists, calendar events, reminder alarms, ... etc, while keeping everything offline.
  • It is  ...
    • It takes notes with sketches, photos. audio ... etc. .
    • It comes with task specific templates and actions that let you get through your daily chores much easier and quicker.
    • It help you keep your documents well organized with file manager and hyperlinks.
    • It mitigate many security risks and privacy issues by staying completely offline and without any third party content.
  • It is not  ...
    • It is not tiny. 
    • It is not social.
    • It is not the easiest to use.
    • It is not a general purpose Web page editor for creating or managing web sites.
