Main Contextmenu

In editing mode, the Main Contextmenu is available in the main toolbar as the Menu () action. It contains most of the editing actions.
  • 
    To close the contextmenu, click on anywhere outside the contextmenu.
Select actions
  • Select an ancestor.
  • Extend selection to the next sibling.
Clipboard actions
  • Copy.
  • Cut.
  • Paste as child.
  • Paste as sibling.
  • Undo action
    ...
    Unfortunately, the access to the browser undo history is limited. So for now, there are two undo actions.
  • Redo the last undo.
  • 
    Show undo history. 
    

  • Undo edit using the undo history.
  • 
    The undo history has a limited size, currently ~99 entries. An action may group multiple changes into a single record or generate multiple records.
  • 
    There is a shortcut for the copy/cut actions. If the current selection is a collapsed selection, it copy/cut the enclosing TR/LI element if there is one. That make it super easy to copy/cut todo items, ... etc.
Attribute actions
  • Edit the tag and attributes of the selected element as raw text.
     

  • Edit styles of the selected element. This action allow editing of the style attribute as raw text. 
    

  • 
    For the gurus, these two actions are all you need to style a document. 
Element actions
  • 
    Move selected element up a level as the next sibling of its parent.
  • 
    Move selected element up a slot as the previous sibling of its previous sibling.
  • 
    Move selected element down a slot as the next sibling of its next sibling.
    • 
      If the current selection is a collapsed selection, the move element up/down actions look up an enclosing TR/LI element which make it super easy to reorder todo items, ... etc. 
  • Clone the the current selection, and insert it as the next sibling. If the current selection is a collapsed selection, it look for an enclosing element with one of the following tags: DL, LI, TR, ARTICLE, SECTION.
  • Wrap selection with an element.
    
  • 
    Unwrap content of the selected element.
  • Insert a paragraph.
    
  • Insert a text node as the next sibling of the element that enclose the current selection. 
  • Insert an element as the first child or next sibling of the selected node. 
    
  • 
    Insert a link (an A element). 
    
  • Insert an image (an IMG element).
    
  • 
    Insert a symbol.
  • 
    Insert a date, a time or both.
  • 
    Insert a list and a list item with symbol bullet.
    • 
      You can change the bullet symbol by clicking on the symbol bullet and bring up the
      Symbol Picker
      . 
  • 
    Insert a list item with symbol bullet to an existing list. Create a new list only if no enclosing list is found.
  • 
    Insert an unordered list and a list item with date string as bullet.
  • 
    Insert list item with date string as bullet.
  • Destruct a widget created by the insert widget action.
    • The current selection must select the top level element of the widget, with one of the following widget marker classes:
    • 
      x-annotation
    • 
      x-flexbox
    • 
      x-pictureframe
    • 
      x-rsb-tab
    • 
      x-slideshow
    • 
      x-sticker
    • Or has one of the following id:
    • 
      x-rightsidepanel
    • 
      x-leftsidepanel
    • 
      x-rightsplitpanel
    • 
      x-leftsplitpanel
  • 
    Insert a widget.

  Character sub-menu 
  • Bold.
  • Italics.
  • Underline.
  • Strike through.
  • Subscript.
  • Superscript.
  • Float left.
  • Float right.

  Paragraph sub-menu 
  • Align left.
  • Align center.
  • Align justify.
  • Align right.
  • 
    Edit alignment
    of selected element.
  • Edit a selected set of styles of the selected element.
  • 
    Edit borders. Click on a side on the outer ring to configure the border. Click on the corners to configure the border radius. Click on the inner ring to edit the box shadow. Click on the center to configure the border and radius on all sides.   
  • Move/Rotate element.
    • 
      The move action only works on element with position:fixed, position:absolute and position:relative that is marked with the x-draggable class, e.g. as created by the sticker widget action.
    • 
      The rotate action use transform and works on any element.
  • 
    Hide selected element. To show the hidden element, use the Outline Panel.
    • 
      The hidden content is shown in the Outline Panel and can be act upon just like another content.
  • Paragraph styles.
  List sub-menu 
  • Insert an ordered list and a list item.
  • Insert an unordered list and a list item.
  • Insert an unordered list and a list item but without bullet.
  • Insert an unordered list and a list item but without bullet and without indent.
  • 
    Insert list item with time string as bullet.
  • Insert list item with text bullet.
    • 
      You can change the date/time string by clicking on the date/time bullet text and bring up the
       Date Picker
      . 
  • Insert a standard HTML list item.
  • 
    Compact layout list style, no margins.
  • 
    Revert compact layout list to normal list margins.
    • 
      Note that some , ,  and  actions always start a new list which enclosed a new list item. The , , ,   and  actions insert a list item to an existing list that enclose the cursor. It create a new list only if there is no enclosing list is found.
