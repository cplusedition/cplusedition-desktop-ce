Search Panel
Introduction
The Global Search Panel perform text search among all the documents using Regular expressions. Regular expression is a powerful tool for text matching. If you are not familiar with it, you can using it for simple text search by avoiding or escaping the regular expression meta characters: .?*+^$()[]{}|. See the java.util.regex.Pattern documentation for details.
