Links Panel
Introduction
The Links Panel shows the links in the document. Broken links are highlighted in red. The HEAD section shows the links to the stylesheets. There are actions to add, remove, rename links. There is also action to open stylesheet for browsing and editing.

Stylesheet Editor
Creating and sharing styles using a stylesheet file, as in the page templates, make styling a document much quicker and easier. Instead of specifying the individual style properties, just specify a CSS class in the class attribute, which apply all the style properties in the CSS class to the attached element. Multiple classes may be specified in a space separated string.
For example the Stylesheet Editor heading above is styled using 
class x-head2 xx-head-orange
 and the tips below uses 
class x-para xx-tips
 Only stylesheet in a writable directory is editable. The  action save the changes to the stylesheet file. The  action close the stylesheet editor without saving changes to the stylesheet file. Changes are applied and reflected in the current document immediately, regardless of whether the changes are saved to the stylesheet file.
