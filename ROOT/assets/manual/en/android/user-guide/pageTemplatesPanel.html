Page Templates
Introduction
The Page Templates Panel contains page templates that can be used to quickly create new documents.
  • 
    Most templates contains placeholders in gray color as hints, eg. the title. Click on it to replace the content and remove the placeholder marker. The background image may also be marked as a placeholder. Click on it to change the background image.
Most page templates also comes with an editing sidebar on the right that host a few template action buttons that are specific for the template.
Templates

The Audio page template starts an audio recording right after the new page is created. Template actions:
  • Add a section.
  • Add an audio recording entry.
  • Remove selected audio recording entry

Blank
The Blank page template creates a minimal blank page without any bells and whistles.
Blog
The Blog page template creates a page for editing blog posts. Template actions:
  • Add a text entry.
  • Add a photo entry.
  • Add an image entry.
  • Show the
    styles menu
    .
  • Remove selected entry.
 Unlike other templates that default to create a unique new page in the current or the drafts directory, the blog template create a new page at
blog/
year/
month/
day.html
 by default. Like the other templates, if the page already exists, it open the existing page instead of creating a new one.
Calendar
The Calendar page template creates a new page for managing monthly calendars. Template actions:
  • Add a section.
  • Add a monthly calendar.
  • Create a pop up annotation over the current selection.
  • Show the
    styles menu
    .
  • Remove selected monthly calendar.
 While you can insert text into the day cells, you may found it more convenient to add events and notes to the day cells using pop up annotations. As you can see there is a template action for that.

  • Hide drawings keeping only the subject header.
  • Remove selected drawing.

The Cards page template creates a stack of index cards. Template actions:

Note 
The Notes page template creates a new page for editing notes. Template actions:

  • Show the
    styles menu

Photo
The Photo page template creates a new page, takes a photo using the camera, import it and embed the image into the new page. Template actions:
  • Add a new roll.
  • Take a photo with the camera and add photo entry.
  • Add an image entry.
  • Show the
    styles menu
    .
  • Remove selected entry/roll.
  • Toggle show/hide caption.

Shopping
The Shopping page template create a page for managing a shopping list. Each entry has a quantity and unit price column. The totals are automatically calculated on changes to the quantity or unit price column or on click at any total or subtotal cells. Template actions:
  • Add a sub-total group.
  • Add an entry, which is also a todo item.
  • Add a photo entry by taking a photo with the camera and insert it as a photo annotation.
  • Show the
    status menu
    .
  • Show the
    styles menu
    .
  • Hide/show the done entries.
  • Remove selected entry.
  •  Cut all items that are not yet done to the clipboard.
  •   Cut all items that are marked as done to the clipboard.
Each entry is also a todo item, click on the status bullet () at the left to toggle the done status.
Todo
The Todo page template creates a new page for managing a todo list. Template actions:
  • Add a section.
  • Add a todo item.
  • Show the
    status menu
    .
  • Show the
    styles menu
    .
  • Hide/show the done entries.
  • Remove an entry.
  •  Cut all todo items that are not yet done ( xx-todo items that are not marked with xx-done) to the clipboard. This is useful for moving the todo items to somewhere else.
  •   Cut all todo items that are marked as done to the clipboard.
Click on the status bullet () at the left to toggle the done status.
