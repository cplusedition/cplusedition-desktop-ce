Outline Panel
  • To select nodes for the actions, you can use the node selection actions ( ,  ) in the
    Main Contextmenu
    , or the
    Main Toolbar
    . But to deal with nodes, your best friend is the
    Outline Panel
    . The Outline Panel shows the hierarchical structure of the element and text nodes. You can cut, paste, insert, delete or edit nodes directly from there which would be reflected in the visual editor.
  • Save the document using the Main Toolbar save (  ) action.
