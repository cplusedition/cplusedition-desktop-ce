HTML Editing
  • Simply type in the new page to insert text. Use the 
    Main Contextmenu ( 

     ) to insert elements and apply styles. There are predefined actions in the Main Contextmenu, eg. insert link (  ), insert image (  ), ... etc, to create most commonly used elements. To create elements that do not has a dedicated action button, use the general purpose insert element (  ), or wrap (  ) actions. For the gurus, the edit attribute (  ) and edit style (  ) actions are all you need to style a document. See the User Guide for details.
