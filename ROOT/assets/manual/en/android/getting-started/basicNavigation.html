Basic Navigation
Files Panel
There are a few options for navigating between documents.
  • At the
    Main Toolbar
    , there is the Home (  ) action that goes to the home page. There are the Back and Forward ( ,  ) actions that goes to the previous or next document in the recent history.
  • You can access all your documents using the
    Files Panel
    . Use the Open (  ) action to open the selected file if it already exists or create a new one using one of the templates. Click on the
    size info
    area of a file entry opens the file directly. Click on a directory entry to go into the directory. To go to the parent directories, use the links in the
    directory path area

    . Use the root (  ) button to go to root directory.
  • At the
    Recents Panel
    , click on a history entry opens the document.
  • In browsing mode, click on any link in the document opens the target document.
