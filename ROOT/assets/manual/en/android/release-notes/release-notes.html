Release notes
for Android
v 2.8.5
  • This version increased the computation required for password encryption and decryption. If you have a low end device, you may notice that backup / restore of keys take longer. Backups created by any previous versions of the app are not affected and should continue to work as before. However, as a general rule, backups created by a particular version of the app should only be used in the same or later version.
  • 
    There is a new Share action () in the Files Panel that export the selected media file (audio, image, PDF), through the Activity Chooser, to an external viewer. Furthermore, the Import/Export actions now works on both the Internal and External backup directories. The Export action can now export HTML and CSS files.
  • 
    Click on a link in editing mode now opens the Edit Link dialog automatically, making it much easier to edit links in the document.
  • 
    There is a new Delete Empty Directories action () in the Files Panel that delete recursively all empty directories under the current directory. That may comes in handy when a lot of empty directories left behind after you move files.
  • 
    Insert/Edit text dialog now use the same font-family as in the document, so it looks very much like how it looks in the document.
  • 
    The search action at the Files Panel now perform simply substring search on filename, ignorecased instead a wildcard search on the filepath, which is easier to use.
  • You can now edit CSS stylesheets while document is in browsing mode.
  • Other minor updates and bugs fixes.
