Style samples
CSS language in stylesheet in form of selector { styles } can apply styles to any element that match the selector. You can often change the look and feel of a whole block with just a single CSS class at the top level element. Here are some examples. To use any of these, simply include samples01.css or copy the corresponding style to your stylesheet.        
xxx-title
A combination of a foreground title style and a glass background style.
<div class="xxx-title-black">Title text</div>
Title text
<div class="xxx-title-white xxx-white-glass">Title text</div>
Title text
<div class="xxx-title-white xxx-smoke-glass">Title text</div>
Title text
xxx-image-row
Layout child images in a single row with equal width using grid layout.
<div class="xxx-image-row"><img src=...>...</div>
