Introduction
The main UI consists of three parts: the main toolbar, the left side panel and the document editing area. The left side panel has three tabs: Global, Explore, and Document. Each of which has a few sub-panels. Toolbar and context menu actions may also shows popup menus and dialogs. The editing area contains the document for editing. The document may has its own sidebar and side panels at the right.
  • Some sidepanels can go translucent using the  action to see what is underneath.
  • Most actions has tooltips, long press on the button would shows the tooltips.
  • Click on the tooltips, warning and error messages prevents it from hiding on timeout. Click again dismiss it before timeout.
Gestures
Interaction with the UI is mostly straight forward point and click. But there are a few special gestures:
  • 
    Fling inward from the left edge at one of the left sidebar tabs opens the left side panel. The vertical position determines which sub-panel would be opened.
    If the current context support turning pages with gestures, eg. in the Calendar Panel, Image Picker, ... etc, fling inward from the left edge goes to the previous page.
  • 
    Fling outward to the left edge goes to the next page if the current context support turning pages with gestures.
  • 
    Fling inward from the right edge opens/edits a CSS file. At the top half, it always open a CSS file picker. At the bottom half, it look for the last editable style sheet linked to the document and edits it if there is one. Otherwise, it opens a CSS file picker. If you select a new CSS file that is not already linked to the document and the document is in editing mode, a link to the CSS file is automatically added to the document. Use the
    Links Panel
    to manage links to css files.
  • 
    Fling outward to the right edge hides/shows the right sidebar.
  • Most dialogs and context menus can be re-positioned by single finger drag at the title bar or the right handle.
  • 
    The drawing canvas has its own set of gestures, see the Drawing Canvas section for details.
