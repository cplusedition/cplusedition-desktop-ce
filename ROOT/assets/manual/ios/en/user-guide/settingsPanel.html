Settings Panel
Introduction
The UI Settings Panel allow customization of the UI.

Most settings should be self-explained except the following:
  • Row height - It specifies the line height of the tree items, eg. file entries in the Files Panel, in percentage of the default line height. Larger value increase the spacing between rows.
  • Image size - It specifies the preferred image dimension for import image actions. For example, if the value of the setting is 1024, then the ImageDimension dialog would suggest to save image with dimension 1024x768 for an image with original size of 2048x1536. You can still change the dimension to other values, it only affects initial value in the dialog.
