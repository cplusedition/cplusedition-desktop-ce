Outline Panel
Introduction
The Outline Panel shows the node tree of the document. It provides a much easier and precise way to select a proper node for actions.
Toolbar
  • Show context menu.
  • Toggle translucent on/off.
  • Extend selection to the next sibling.
  •  Note that you may long press at a sibling to extend the selection instead of pressing this button repeatedly.
  • Expand all.
  • Collapse all.
Contextmenu
The Outline Panel context menu is a subset of the Main context menu. The action looks and works, mostly, like those in the Main context menu. There are a few Outline Panel specific actions:
  • Show initial selection when the panel is opened.
  • Toggle show/hide whitespace nodes.
  • Toggle ignore case for search in the Outline Panel.
  • Show the toolbar/search bar.
  • Delete the selection. This is like cut () but without saving the deleted content to the local clipboard.
  • Show/hide of an element.
    • 
      Note that hidden elements are not visible in the visual editor, so you has to select or show it here.
