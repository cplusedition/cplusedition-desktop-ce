Selection
  • Most actions require a selection. Some actions, eg. insert image (), require a single insertion point | (a collapsed selection). Some actions, eg. unwrap (), require an element selection and some other actions, e.g. insert link (), work on both. Depending on the current selection, some actions may not be available.
